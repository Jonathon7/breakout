Drawing on the Canvas

We will be using only two shapes on the canvas to create breakout. Here is an example of how to draw a rectangle:

ctx.beginPath() discards any old path and indicates that you are starting a new path for the shape that is being drawn. The .rect() function allows you to draw four lines and then fill those lines with a color. The first two parameters in the function are the top and left coordinates of the shape. The second two parameters are the width and height of the shape in pixels. The fillStyle() and fill() functions will fill the shape with a color. These two functions only work on a closed path otherwise the color will leak and fill the entire canvas.

Drawing a Circle

This example is the same as before except we are using the arc() function to draw the shape. This function takes 6 parameters. The first two are the coordinates for where the shape is going to be drawn on the canvas. The third parameter is the radius of the arc. The next two parameters are the start angle and the end angle of the arc. So in order to draw a full circle you have to use the Math.PI function. The last parameter is omitted from the example (optional) but it takes a boolean value of true or false. False will draw the arc clockwise and true (default) will draw the arc counter-clockwise.
